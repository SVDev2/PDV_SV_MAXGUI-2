PDV SV MAXGUI-2

Política de Privacidade

Data deste termo: 06 de março de 2026

1. Introdução

Esta Política de Privacidade descreve como o sistema PDV SV MAXGUI-2 coleta, utiliza, armazena e protege informações dos usuários. Nosso objetivo é garantir transparência, segurança e respeito à privacidade, seguindo boas práticas de proteção de dados e os princípios da legislação brasileira. Ao utilizar o sistema, o usuário declara estar ciente desta política.

2. Responsável pelo sistema

  • Responsável pelo sistema: SER VIVO
  • Sistema: PDV SV MAXGUI-2

3. Dados coletados

  • Dados de autenticação
  • Nome de usuário
  • Senha protegida
  • Identificação de sessão

Esses dados são utilizados para controle de acesso ao sistema. Dados de uso do sistema O sistema pode registrar:

  • data e horário de acesso
  • frequência de acesso
  • tempo de sessão
  • páginas utilizadas
  • funcionalidades acessadas

Esses dados são usados para melhorar o sistema e aumentar a segurança.

Dados operacionais O sistema também pode armazenar dados relacionados às operações realizadas, como:

  • vendas
  • produtos cadastrados
  • relatórios administrativos
  • registros operacionais do PDV

4. Uso de cookies

O sistema pode utilizar cookies e tecnologias similares para:

  • manter o usuário autenticado
  • lembrar preferências de login
  • registrar sessões de uso
  • coletar estatísticas de utilização

Cookies são pequenos arquivos armazenados no navegador do usuário. O usuário pode configurar seu navegador para bloquear cookies, porém algumas funcionalidades do sistema podem não funcionar corretamente.

5. Segurança dos dados

São adotadas medidas de segurança para proteger os dados armazenados, incluindo:

  • controle de acesso
  • autenticação de usuários
  • proteção do banco de dados

Apesar disso, nenhum sistema é totalmente livre de riscos.

6. Direitos do usuário

Conforme a Lei Geral de Proteção de Dados Pessoais (LGPD), o usuário pode solicitar:

  • acesso aos seus dados
  • correção de informações
  • exclusão de dados quando aplicável

7. Validade

Este documento é válido apenas para a última versão do site

  • versão do site para este contrato: 0.3.2 Alpha
